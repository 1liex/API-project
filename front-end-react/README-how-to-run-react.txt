طيب هنا بشرح لك كيف تقدر تشغل مشروع رياكت

اول شي انت محتاج انك تحمل (node js)


و هذا راطب التحميل
(https://nodejs.org/en/download)

حمله و اتبع خطوات التحميل

بعد كذا تحتاج تتاكد اذا تحمل او لا من خلال الامر ذا
(node --version)

المفروض تطلع لك نسخة التحميل حقتك زي كذا او شي مشابه لكذا
(v20.15.1)

بعد ما تحمل node js


كل الي تحتاجة الان هو انك تخش على ملف front-end-react

من خلال الامر ذا في التيرمنل cd front-end-react
بعدها تحتاج انك تحمل الباكيجز الخاصه بالمشروع من خلال الامر ذا في التيرمنل

اختار واحد فيهم
npm i / npm install



وبعدها تكتب ذا الامر في التيرمنل
npm run dev

و بس بيطلع لك في التيرمنل رابط اضغط عليه و بيوديك لصفحة المشروع في المتصفح


و لو كنت حاب مثلا تسوي مشروعك الخاص من باب الفضول بكل بساطة سوي لك مجلد في سطح المكتب و سميه ايش ما تبغى

بعدين خش على الملف و اكتب في التيرمنل

npm create vite@latest name-of-ur-project -- --template react
بتلاحظ انه تم انشاء مجلد بالاسم الي انت مسميه فيه 

بعدها بكل بساطة عشان تشغل المشروع ادخل على الملف بذا الامر في التيرمنل 

cd name-of-ur-project

بعدها تكتب في التيرمنل

npm run dev
